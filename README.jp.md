# 料理手順動画における動詞翻訳データセット

© 2026, Itsuki Takefuji, Kazuyoshi Takata, Isao Goto, and Takashi Ninomiya, Ehime University

本リポジトリでは，料理手順動画における英語動詞の日本語翻訳を評価するためのデータセットを公開しています。

## 概要

本データセットは，料理手順に含まれる多義的な英語動詞の翻訳を評価することを目的として構築したものです。

料理手順では，英語動詞の適切な日本語訳が，テキスト情報だけでなく，食材の状態，調理器具，実際の動作過程などに依存する場合があります。例えば，*cook*，*boil*，*fry*，*spread*，*toss* などの動詞は，文脈や映像中の動作によって異なる日本語訳が適切になることがあります。

本データセットは，機械翻訳，マルチモーダル機械翻訳，料理手順理解，および動画情報を用いた語義曖昧性解消の研究に利用することを想定しています。

## データセットの説明

本データセットには，料理手順動画の各区間に対応する英語文，日本語参照訳，動画ID，開始時刻，終了時刻が含まれています。

データはCSV形式で提供しています。

## ファイル構成

```text
train.csv
test.csv
```

各CSVファイルには，以下の列が含まれています。

| 列名         | 説明          |
| ---------- | ----------- |
| `en`       | 英語の料理手順文    |
| `ja`       | 日本語の参照訳     |
| `video_id` | YouTube動画ID |
| `start`    | 対象区間の開始時刻   |
| `end`      | 対象区間の終了時刻   |

## 注意事項

本データセットには，動画ファイル本体は含まれていません。

また，本データセットは元動画，英語キャプション，および元アノテーションの著作権を所有するものではありません。元動画の著作権は，それぞれの権利者に帰属します。元動画や関連データを使用する場合は，各データセットおよび動画提供元の利用条件，著作権情報を確認してください。
